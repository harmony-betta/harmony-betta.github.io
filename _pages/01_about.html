Harmony Framework

Harmony Framework is a PHP framework built under the Slim Framework. Harmony is a unit of some famous packages that have been widely used by developers in the world.

Initially Harmony Framework was built for personal purposes to facilitate my work as a website developer, but I also want Harmony Framework can be useful for other developers.

Equipped with Artisan though still limited; create controllers, middleware, create database migrations, etc. with easy routing as used by the Slim Framework.

Harmony Framework uses Twig Templating to build views, equipped with Twig Helpers that you can add yourself.
